# Pizza Ordering Application üçï

A simple Pizza Ordering Application built using Python Tkinter.

## Features
- Customer name & delivery address input
- Choose pizza size (Small / Medium / Large)
- Select toppings (Pepperoni, Mushrooms, Onions)
- Order summary popup message

## Tech Stack
- Python
- Tkinter (GUI)

## How to Run
1. Download/Clone this repository
2. Make sure Python is installed
3. Run the project:
   python pizza_ordering_app.py
